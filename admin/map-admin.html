📖 功能说明

该功能是管理元宇宙主地图上的热区标注，用户端在主地图上点击热区可跳转到对应场景或查看装饰元素

一、地图管理

  • 上传地图：点击"更换地图图片"上传主地图底图，支持 PNG/JPG 格式
  • 缩放配置：设置地图缩放比例（100%~300%），控制用户端地图的拖拽范围

二、热区标注

  • 拖拽绘制：在地图上拖拽生成矩形热区区域，实时预览位置和大小
  • 三种热区类型：
    • 📍跳转热区：点击跳转到指定场景图片，需关联跳转目标
    • 🎨装饰热区：纯视觉装饰元素，配置子类型和参数
    • 🚶起始位置：角色初始站位，自动填入名称"人物起始位置"
  • 装饰子类型（4种）：
    • 🖼️雪碧图动画：上传雪碧图 + 坐标JSON（可选）+ 帧数 + 帧速
    • 🎬Lottie动画：上传 .lottie 文件
    • ✨CSS动效：上传图片 + 选择效果（脉冲/浮动/摇摆/光晕/旋转/弹跳）
    • 🌟原生粒子：选择粒子类型 + 生成率
    • 均支持宽度(px) 和 Z轴序号配置

三、热区列表

  • 按类型分三列展示：跳转热区 / 装饰热区 / 起始位置
  • 每列显示热区名称、类型标识、Z轴序号、宽度、坐标位置
  • 支持编辑和删除热区
